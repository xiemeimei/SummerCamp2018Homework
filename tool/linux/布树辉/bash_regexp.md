## regexp tips

^                       - begining of the string
$                       - end of the string

()                      - group 找到的匹配
[ ]*                    - 任意个数的空格
.*                      - 任意长度的内容
[^ ]                    - 表示非空格的字符
[0-9]{11}               - 表示有11个字符的数字

${BASH_REMATCH[1]}      - 第一个匹配 （从下标1开始）
